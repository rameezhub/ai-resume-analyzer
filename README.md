 AI Resume Analyzer

A Flask-based web app to analyze resumes and calculate ATS score.

 Features
- Upload resume PDF
- ATS score calculation
- Skill matching
- Missing skills detection
- Suggestions
- Charts
- Dark UI

 Tech Stack
- Python
- Flask
- PyPDF
- HTML, CSS
- Chart.js

 How to Run
```bash
pip install flask pypdf
python app.py
